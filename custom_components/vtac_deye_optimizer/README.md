# VTAC Deye Optimizer

This integration is configured from the Home Assistant UI.

Do not add a `vtac_deye_optimizer:` block to `configuration.yaml`.

After installing or updating through HACS, restart Home Assistant and add it from:

`Settings -> Devices & services -> Add integration -> VTAC Deye Optimizer`

The setup form is pre-filled with the Solarman entities for your VTAC battery and
Deye inverter.
